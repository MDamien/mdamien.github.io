Modifications directes dans le navigateur

Il est possible d'exécuter du JavaScript dans un navigateur en entrant son propre code. Faisons un petit essai.
Allez sur un site internet contenant du texte, comme Wikipedia (Pokémon).
Clic droit dans le navigateur et sélectionnez une option "inspecter l'élément" ou similaire. Allez ensuite dans l'onglet console. Nous entrerons du code dans le champ de la console.

Modifier la couleur du texte des paragraphes

Il suffit de sélectionner tous les éléments p et de modifier leur attribut style : 


<script>
paragraphes = document.getElementsByTagName('p');
for (i = 0; i < paragraphes.length; i++) {
	p = paragraphes[i];
	p.style['color'] = 'blue';
}
</script>
