Mise en pratique

Nous partons d'un fichier HTML contenant une balise <video>, le script popcorn.js et un exemple d'utilisation

<!doctype html>
<html>
  <head>
    <script src="http://popcornjs.org/code/dist/popcorn-complete.min.js"></script>
  </head>
  <body>
    <video height="180" width="300" id="ourvideo" controls>
      <source src="http://videos.mozilla.org/serv/webmademovies/popcornplug.ogv">
    </video>
    <h1 id="footnotediv"></h1>
    <script>
         var pop = Popcorn("#ourvideo");
         pop.footnote({
           start: 2,
           end: 6,
           text: "Pop!",
           target: "footnotediv"
         });
         pop.play();
    </script>
  </body>
</html>

Affichez :

  • Une note de bas de page "¯\(◉‿◉)/¯" à t=1s via pop.footnote()

  • Un sous-titre "你好" à t=2s via pop.subtitle()

  • Un popup affichant "(◕‿◕✿) < FIN" (utilisant alert() ) à t=6s

    },

pop.play();
    </script>

Ajoutez un bouton "play/pause" pour mettre en pause la vidéo si elle est en cours de lecture ou en lecture si elle est en pause
Utilisez pop.pause(), pop.play() et pop.playing()
<button onclick="play_or_pause()">Play/Pause</button>
<script>
function play_or_pause(){
    if(pop.paused()){
        pop.play();
    }else{
        pop.pause();
    }
}
</script>
Création d'un plugin
A l'aide de la documentation, creez un plugin christmas qui permettra d'entourer la vidéo par une aura rouge et qui s'utilisera comme cela :
pop.christmas({
    start:5,
    end:10,
})
Laissez libre court à votre imagination pour l'effet que votre plugin va déclencher.
Pour accéder à l'élément vidéo, utilisez this.media
Pour l'aura, on pourra utiliser un style css tel que : box-shadow: 0px 0px 60px red ;
Popcorn.plugin( "christmas" , {
      start: function( event, options ){
          this.media.style['box-shadow'] = '0px 0px 60px red';
      },
      end: function( event, options ) {
          this.media.style['box-shadow'] = '';
      },
  });
var p = Popcorn( "#video" ).play().christmas({start: 1,end: 2,});
