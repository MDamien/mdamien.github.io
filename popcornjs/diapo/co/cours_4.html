Plugins

Popcorn.js possède également des plugins. Ces derniers permettent de faire facilement des interactions supplémentaires.

Plugin Subtitle : Ajouter des sous-titres

pop.subtitle({
    start: 5,
    end: 15,
    text: "All your base are belong to us",
});

Plugin Image : Ajouter une image

pop.image({
    start: 5,
	  end: 15,
	  src: "http://i.imgur.com/BTNIDBR.gif",
});

Plugin Footnote : Ajouter une note de bas de page

 pop.footnote({
   start: 2,
   end: 6,
   text: "Ceci n'est pas une note de bas de page",
   target: "footnotediv"
 });

Plugin Wikipedia : Afficher une page Wikipedia

pop.wikipedia({
    start: 0,
    end: 10,
    src: "http://en.wikipedia.org/wiki/Pokemon",
    title: "More about these 'Pokemons'",
    target: "wikidiv"
});

Plugin Code : Exécuter du code arbitraire

pop.code({
   start: 1,
   end: 3,
   onStart: function( options ) {
       document.getElementById("my_div").innerHTML = "On";
   },
   onEnd: function( options ) {
       document.getElementById("my_div").innerHTML = "Off";
   }
});

Création de plugin

    Popcorn.plugin("plugin_name", {
        start: function(event, options) {
            console.log('start');
        },
        end: function(event, options) {
            console.log('end');
        },
    });
