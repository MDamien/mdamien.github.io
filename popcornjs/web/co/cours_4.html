Plugins

Popcorn.js possède également des plugins. Ces derniers permettent de faire facilement des interactions supplémentaires.

Exemple : Plugin Subtitle : Ajouter des sous-titres

1
pop.subtitle({
2
    start: 5,
3
    end: 15,
4
    text: "All your base are belong to us",
5
});
pop.subtitle({
    start: 5,
    end: 15,
    text: "All your base are belong to us",
});

Exemple : Plugin Image : Ajouter une image

1
pop.image({
2
    start: 5,
3
	  end: 15,
4
	  src: "http://i.imgur.com/BTNIDBR.gif",
5
});
pop.image({
    start: 5,

Exemple : Plugin Footnote : Ajouter une note de bas de page

1
 pop.footnote({
2
   start: 2,
3
   end: 6,
4
   text: "Ceci n'est pas une note de bas de page",
5
   target: "footnotediv"
6
 });
 pop.footnote({
   start: 2,
   end: 6,
   text: "Ceci n'est pas une note de bas de page",
   target: "footnotediv"
 });

4

5
    title: "More about these 'Pokemons'",
6
    target: "wikidiv"
7
});

Complément : Création de plugin

Vous pouvez aussi très facilement créer des plugins

1
    Popcorn.plugin("plugin_name", {
2
        start: function(event, options) {
3
            console.log('start');
4
        },
5
        end: function(event, options) {
6
            console.log('end');
7
        },
8
    });
    Popcorn.plugin("plugin_name", {
        start: function(event, options) {
            console.log('start');
        },
        end: function(event, options) {
            console.log('end');
        },
    });
