Évènements JavaScript

JavaScript détecte des "évènements" pour déterminer s'il doit exécuter du code.
Un évènement peut par exemple être un clic de souris, le survol d'un élèment ou l'appui sur une touche de clavier.

Noms de quelques évènements utilisés par JavaScript

onclick : Clic sur un élément.
onmouseover : Survol d'un élément.
onmouseout : Fin du survol d'un élément.
onkeydown : Appui sur une touche de clavier.
onkeyup : Fin de l'appui sur une touche de clavier.

HTML : éléments qui seront utilisés par JavaScript

Pour savoir si il doit exécuter du code, JavaScript attend des évènements, mais dans un certain contexte.
On regarde le plus souvent quel élément HTML a reçu l'évènement (quel bouton a reçu un clic, quel champ de texte reçoit des entrées de clavier...).

Détection simple : nom de tags

On recherche directement si les éléments ont reçu un évènement :
<button>Bouton</button>
<p>Paragraphe</p>

Détection par identifiant

On cherche un identificant HTML particulier, identifié par un attribut id :
<button id="bouton-1">Bouton 1</button>
<button id="bouton-2">Bouton 2</button>
