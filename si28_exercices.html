Évènements JavaScript

JavaScript détecte des "évènements" pour déterminer s'il doit exécuter du code.
Un évènement peut par exemple être un clic de souris, le survol d'un élèment ou l'appui sur une touche de clavier.

Noms de quelques évènements utilisés par JavaScript

onclick : Clic sur un élément.
onmouseover : Survol d'un élément.
onmouseout : Fin du survol d'un élément.
onkeydown : Appui sur une touche de clavier.
onkeyup : Fin de l'appui sur une touche de clavier.

HTML : éléments qui seront utilisés par JavaScript

Pour savoir si il doit exécuter du code, JavaScript attend des évènements, mais dans un certain contexte.
On regarde le plus souvent quel élément HTML a reçu l'évènement (quel bouton a reçu un clic, quel champ de texte reçoit des entrées de clavier...).

Détection simple : nom de tags

On recherche directement si les éléments ont reçu un évènement : 

<button>Bouton</button>
<p>Paragraphe</p>

Détection par identifiant

On cherche un identificant HTML particulier, identifié par un attribut id : 

<button id="bouton-1">Bouton 1</button>
<button id="bouton-2">Bouton 2</button>

Détection par classe

Les classes sont utilisées pour identifier des groupes d'éléments HTML. Tout comme en CSS, il est possible d'utiliser l'attribut class des éléments HTML pour les sélectionner : 

<button class="bouton-bleu">Bouton 1</button>
<button class="bouton-bleu">Bouton 2</button>
<button class="bouton-rouge">Bouton 3</button>
<button class="bouton-rouge">Bouton 4</button>

Exemples en JavaScript

Le code JavaScript se fait en 2 étapes pour chaque évènement :
1. Récupération de l'élément HTML dans une variable.
2. Liaison de cet élément à un/des évènements. Liaison de l'élément à une fonction pour chaque évènement.

Détection par identifiant

Pour obtenir un élément par son identifiant, on utilise 

document.getElementById('nom-identifiant');

<script>
    //1 : Lier bouton1 à l'élément HTML avec id="nom-identifiant"
    var bouton1 = document.getElementById('bouton-1');
    //2 : Liaison d'un évènement (onclick) à cet élément HTML
    	bouton1.onclick = function() {
    		//Code à exécuter après click sur bouton1
	}
</script>

Détection par classe (plusieurs éléments retournés)

Pour obtenir les éléments ayant une certaine classe, on utilise 

document.getElementsByClassName('nom-classe');

<script>
	//1 : Récupération de TOUS les éléments ayant bouton-bleu comme classe.
	var boutonsBleus = document.getElementsByClassName("bouton-bleu");

	//2 : Liaison d'une fonction pour chaque bouton-bleu.
	// Se déclenche au survol d'un bouton-bleu
    	for (var i = 0; i < boutonsBleus.length; i++) {
    		var boutonBleu = boutonsBleus[i];

    		boutonBleu.onmouseover = function() {
    			//Code à exécuter après survol sur bouton-bleu
    		}
	}
</script>
